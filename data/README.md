# サンプルデータセット

このディレクトリには、アプリケーションの機能をテストするためのサンプルデータセットが含まれています。

## データセット一覧

### 1. sample_survey.csv - アンケートデータ
**用途**: アンケート分析、記述統計、クロス集計、NPS分析

**列の説明**:
- `respondent_id`: 回答者ID
- `age`: 年齢
- `gender`: 性別
- `region`: 地域
- `q1_satisfaction`: 満足度（1-5）
- `q2_quality`: 品質評価（1-5）
- `q3_price`: 価格評価（1-5）
- `q4_service`: サービス評価（1-5）
- `q5_recommendation`: 推奨度（1-5）
- `nps_score`: NPS（ネットプロモータースコア）（0-10）
- `feedback`: フィードバック（自由記述）

**サンプル数**: 50件

**分析例**:
- リッカート尺度分析
- NPS計算
- 地域別・性別別のクロス集計
- 満足度の相関分析

---

### 2. sample_purchase_log.csv - 購買ログデータ
**用途**: 購買分析、RFM分析、バスケット分析、時系列分析

**列の説明**:
- `transaction_id`: 取引ID
- `customer_id`: 顧客ID
- `purchase_date`: 購入日
- `product_category`: 商品カテゴリー
- `product_name`: 商品名
- `quantity`: 数量
- `unit_price`: 単価
- `total_amount`: 合計金額
- `payment_method`: 支払い方法

**サンプル数**: 50件

**分析例**:
- RFM分析（最終購入日、購入頻度、購入金額）
- カテゴリー別売上分析
- 顧客セグメンテーション
- 時系列トレンド分析

---

### 3. sample_text_data.csv - テキストデータ
**用途**: テキスト分析、感情分析、トピックモデリング、単語頻度分析

**列の説明**:
- `id`: テキストID
- `category`: カテゴリー
- `sentiment`: 感情（ポジティブ/ネガティブ/中立）
- `text`: テキスト内容（日本語）

**サンプル数**: 50件

**分析例**:
- 単語頻度分析
- ワードクラウド生成
- トピックモデリング（LDA）
- 感情別の分類
- カテゴリー別の傾向分析

---

## 使用方法

1. アプリケーションのファイルアップロードページで「サンプルデータを使用」を選択
2. 使用したいサンプルデータセットを選択
3. 「サンプルデータを読み込む」ボタンをクリック

または、これらのCSVファイルをダウンロードして、ファイルアップロード機能でアップロードすることもできます。

## 注意事項

- これらのデータは分析機能のデモンストレーション目的で作成された架空のデータです
- 実際のビジネスデータではありません
- 自由に編集・拡張して使用できます
